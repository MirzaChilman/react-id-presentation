# What is smart and dumb components

---

# Smart components

<ul>
  <Appear>
    <li>Focus on how things <span style={{color:"red"}}>work</span>ğŸ’»</li>
    <li>Contain business logic</li>
    <li>Provide data</li>
    <li>Using class or generated using HOC</li>
    <li>Usualy full of state</li>
  </Appear>
</ul>

---

# Dumb components

<ul>
  <Appear>
    <li>Focus on how things <span style={{color:"red"}}>look</span>ğŸ’»</li>
    <li>Does not have business logic</li>
    <li>Do not care how data is loaded, mutated or processed</li>
    <li>Written in functional components</li>
  </Appear>
</ul>
